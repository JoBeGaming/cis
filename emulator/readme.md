## Emulator for CiS (CPU in Survival)

# Requirements
Requires Python, and knowledge of the assembly code of CiS (see down below).

# Usage:
There are 3 tabs in the main code space (on the left half).
`Code`, `Memory`, and `Settings`.

On the I/O code space (on the right half), there will be a display, a 7 segment display, and 2 controllers.

# Code
In `Code`, you can type in your own code in the textbox below.
After you are finished, press the `Run` button above the textbox.
The code inputted in the textbox will be ran and the output will be shown in the I/O code space.

# Memory
In `Memory`, you can see all the memory of the CPU.
It has the Registers, RAM, and the Call Stack.

# Settings
In `Settings`, you can edit settings for the CPU such as the clock speed.
